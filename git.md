2.git使用分支<br>
  __查看分支__git branch<br>
  __创建分支__git branch <name> <br>
  __切换分支__git checkout <name>  <br>
  __创建+切换分支__ git checkout -b <name> <br>
  __合并某分支到当前分支__ git merge <name> <br>
  __删除分支__  git branch -d <name> <br>
